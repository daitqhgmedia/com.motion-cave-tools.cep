PingPong Expression Tester

Instructions:

  1. Open an After Effects project
  2. Select a Composition
  3. Select one or more Layers
  4. Select properties (Position, Rotation, Scale, etc.)
  5. Make sure the properties have Keyframes
  6. Click "Run PingPong Script" button

Description: This script adds loopOut("pingpong") expressions to all selected properties in your After Effects project. The pingpong loop expression causes animations to play forward and backward continuously.
